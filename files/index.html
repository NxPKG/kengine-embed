Kengine embedded file server works!

Make sure to replace the contents of this folder with your own files.

You will need to clone this Kengine plugin and replace the files in the files subfolder, then build Kengine with your locally-cloned plugin to embed your own content. Something like this: 

$ git clone https://github.com/nxpkg/kengine-embed.git
$ cd kengine-embed
$ rm files/*
$ cp -r ~/site files/
$ xkengine build --with github.com/nxpkg/kengine-embed=.

Then your kengine binary has your site's files embedded with in it. Be sure to specify it in your config: 

file_server {
	fs embedded
}
